How Pitia Works

  • 1. Send to our systems the users classifications using us REST API
  • 2. Our systems will store the user information in a safety and private way in order to use it to improve your future recommendations
  • 3. Based on the previous provided classifications, the system generates a list of the best elements to be recommended for this user
That's all!, Pitia is a super easy way to recommend items to your customers based in one of the most accuarates recommendation algorithms, we will store the data in a safety way, process it and generate recommendations on the fly!
Pitia is able to scale according to your present and future necessities.

High Availability

Each group contains at least 2 shards, and each shard is allocated on a different physical machine in order to assure that if one of us servers goes down your coustomers will continue receiving the best service. That's why always the first shard in a group is composed by two shards being the same price by the first two than by a single shard.
All the provided recommendations are persisted in a safe storage system apart of us data centers in order to warranty that in the worst of the situations we can have the systems up and running with the last provided information in question of minutes.
